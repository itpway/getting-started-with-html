Basics of the Web & HTML

The Web is a collection of documents made up of HTML (HyperText Markup Language).
Invited in the 1990's
HTML is the main type of documents on the web

All of these types of files can be found on the web
Plain Text
HTML
Links
Videos
Music

Major Components of the web are
*You
*Your computer
*The Internet (different browsers - Chrome Firefox, IE)
*HTTP (Protocol)
*Servers (Sitting in a closet hosting files)

HTML is made up of the following
Text Content - What you can see
Markup - What it looks like
References to anything like videos, links, music, etc

Markups are made up of tags
*b - Bold
*br - Line break
*em - Emphasis (Italics)
*p - paragraph
*a - Anchors
*img - images
*span - acts like a br
*div - Wraps the text in an invisible box
